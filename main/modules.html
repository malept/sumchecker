default

  • default(algorithm: string, checksumFilename: string, baseDir: string, filesToCheck: string | string[]): Promise<void>
    • Defined in src/index.ts:241

    Validates the checksum of files from a given checksum file, usually generated by programs such as sha256sum. Throws an error if any file to check specified fails validation.

    Usage

     
    const sumchecker = require("sumchecker");

    // NB: Top-level await is available in Node.js >= 14.8.0. Non-top-level-await syntax is left as an
    // exercise to the reader.
    try {
      await sumchecker(algorithm, checksumFilename, baseDir, filesToCheck);
      console.log("All files validate!");
    } catch (error) {
      console.error("An error occurred", error);
    }

    Parameters

    • algorithm: string

      The hash algorithm used in checksumFilename. Corresponds to the algorithms allowed by crypto.createHash().

    • checksumFilename: string

      The path to the checksum file.

    • baseDir: string

      The base directory for the files specified in filesToCheck.

    • filesToCheck: string | string[]

      One or more paths of the files that will be validated, relative to baseDir.

    Returns Promise<void>
